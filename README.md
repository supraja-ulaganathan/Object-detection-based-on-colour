# Object-detection-based-on-colour
Object detection system that identifies and tracks objects based on their color attributes. Includes a main detection script and a separate module for color identification. It provides practical use of HSV color space and OpenCV for computer vision tasks.
# Features
- Detects and tracks objects using color segmentation  
- Real-time webcam feed support  
- Separate module for color identification
# Tools & Technologies
- Programming Language: Python  
- Libraries: OpenCV, NumPy  
- Concepts: HSV color space, contour detection, image moments
